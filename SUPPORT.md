# Support

Thanks for using MAL UI. This project is maintained by MAL Devs and community
contributors.

## Where to Ask

- Bugs: open a bug report with a minimal reproduction.
- Feature ideas: open a feature request and describe the user problem.
- Documentation gaps: open a documentation issue or pull request.
- Security issues: follow `SECURITY.md`; do not disclose vulnerabilities in
  public issues.

## Before Opening an Issue

Please check:

- The README and docs.
- Existing open and closed issues.
- The latest package version.
- Whether the issue reproduces in a small example.

## Maintainer Capacity

This is an open-source project. Maintainers review issues and pull requests as
time allows. Clear reproductions, focused changes, and passing tests make help
much faster.
