# Changelog

All notable changes to this project are documented here.

This project follows Semantic Versioning and uses Conventional Commits for
commit messages. Maintainers should update this file as part of every release.

## [Unreleased]

### Added

- Public open-source community files for contributions, support, security, and
  governance.
- GitHub issue templates and pull request template for consistent triage.
- Project documentation for structure, accessibility expectations, and releases.

## [0.1.5]

### Added

- Mantine v9 based `mal-ui` package with tree-shakeable subpath exports.
- MAL UI theme tokens and `malTheme`.
- Bundled global stylesheet exposed as `mal-ui/styles.css`.
- Bun build pipeline for JavaScript, CSS, and TypeScript declarations.
- Next.js demo app covering every public subpath.
